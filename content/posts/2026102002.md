### 接入Java示例——改一行代码就行

真的只需要改一行代码。下面是一个完整的Java示例，让你快速体验o3的推理能力：

**1. 引入依赖（Maven）**

xml
<dependency>
    <groupId>com.theokanning.openai-gpt3-java</groupId>
    <artifactId>service</artifactId>
    <version>0.18.2</version>
</dependency>


**2. Java代码示例**

java
import com.theokanning.openai.OpenAiService;
import com.theokanning.openai.completion.chat.ChatCompletionRequest;
import com.theokanning.openai.completion.chat.ChatMessage;
import java.util.Arrays;

public class O3Demo {

    public static void main(String[] args) {
        // 关键修改：替换base_url为千聚API
        String apiKey = "你的千聚API聚合站Key";
        String baseUrl = "https://www.qianjuai.com/v1";  // 千聚API聚合站地址
        
        OpenAiService service = new OpenAiService(apiKey, baseUrl, Duration.ofSeconds(60));

        // 构建o3请求（注意：o3支持reasoning_effort参数）
        ChatCompletionRequest request = ChatCompletionRequest.builder()
            .model("o3-mini")  // 可换为o3, gpt-4o等
            .messages(Arrays.asList(
                new ChatMessage("system", "你是一个顶级Java架构师，擅长代码分析与优化。"),
                new ChatMessage("user", "请详细分析以下Java代码的潜在问题，并给出优化建议：\n" +
                    "\"\"\"java\n" +
                    "class Calculator {\n" +
                    "    public int add(int a, int b) {\n" +
                    "        return a + b;\n" +
                    "    }\n" +
                    "}\n" +
                    "\"\"\"")
            ))
            .maxTokens(2048)  // o3推荐使用max_completion_tokens或max_tokens
            .build();

        // 流式输出或非流式
        service.createChatCompletion(request).getChoices().forEach(choice -> {
            System.out.println(choice.getMessage().getContent());
        });
    }
}


**避坑代码片段**：

- **坑1**: o3模型要求使用`max_completion_tokens`参数而非`max_tokens`，旧库可能默认用`max_tokens`，建议使用最新版本（如0.18.2及以上）的服务库，或在请求构建时显式设置。
  
java
// 正确的o3请求构建（兼容旧库）
.request.setMaxTokens(2048);  // 某些旧库会自动转成max_completion_tokens


- **坑2**: o3对System Prompt非常敏感，不要写太长，否则容易丢失输出质量。建议将关键指令放在User Message中。

- **坑3**: 千聚API聚合站默认支持并发无限制，但有速率限制（QPS），建议使用`Semaphore`或`RateLimiter`避免被限流。

java
// Guava的RateLimiter
RateLimiter limiter = RateLimiter.create(10.0); // 每秒10个请求
limiter.acquire(); // 获取许可
