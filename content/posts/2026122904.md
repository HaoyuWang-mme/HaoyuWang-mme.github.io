## Node.js 调用示例对比

### 官方版（直接调用 Anthropic SDK）

javascript
// 安装：npm install @anthropic-ai/sdk
import Anthropic from '@anthropic-ai/sdk';

const anthropic = new Anthropic({
  apiKey: 'sk-ant-...',  // 需海外信用卡注册
  baseURL: 'https://api.anthropic.com', // 需代理访问
});

async function callClaude() {
  const msg = await anthropic.messages.create({
    model: 'claude-3-haiku-20240307',
    max_tokens: 1000,
    messages: [{ role: 'user', content: 'Hello' }],
  });
  console.log(msg.content);
}


问题：需要海外网络、绑卡、美元计费。`claude-3-haiku` 官方定价 0.25 美元/百万输入，0.125 美元/百万输出（以 Anthropic 官网为准），加上代理费用，成本不低。

### 社区版（用公开的免费 Key）

javascript
// 使用第三方免费代理，接口格式未必兼容
const response = await fetch('https://some-free-proxy.com/v1/messages', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'free_key_123',
  },
  body: JSON.stringify({ model: 'claude-3-haiku', ... }),
});


优点：零成本。缺点：Key 随时失效、并发限制大、不保证数据安全（可能被记录请求内容）。做个人玩具可以，生产环境绝对不敢用。

### 魔改版（自己搭建中转服务器）

javascript
// 在自己服务器上部署了 One API，对外开放接口
const client = new Anthropic({
  apiKey: 'my-custom-key',
  baseURL: 'https://my-server.com/v1', // 需自己维护服务器
});


优点：可控，成本低（仅服务器费）。缺点：需要懂运维、服务器挂掉影响服务、流量费+磁盘费、还得解决国内访问海外 API 的代理问题（又要买代理）。整体隐性成本很高。
