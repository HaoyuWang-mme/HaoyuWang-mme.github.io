## 第4步：一行代码，调通你的第一个模型

这里以Python为例。假设你之前用的是OpenAI官方的库，接入千聚只需要改**两行参数**：

python
from openai import OpenAI

# 原来调用OpenAI：
# client = OpenAI(api_key="你的OpenAI Key", base_url="https://api.openai.com/v1")

# 现在接入千聚：
client = OpenAI(
    api_key="sk-你刚才复制的千聚Key",  # 替换成你的Key
    base_url="https://www.qianjuai.com/v1"   # 替换成千聚接口地址
)

response = client.chat.completions.create(
    model="gpt-4o",  # 模型名直接用官方名字，千聚会自动映射
    messages=[{"role": "user", "content": "你好，用一句话介绍一下千聚AI中转站"}],
    stream=False
)

print(response.choices[0].message.content)

运行这段代码，如果看到模型回复了内容，恭喜你——接入成功！

**💡 常见报错与解决：**
- **401错误（认证失败）：** 检查API Key是否复制错误，或者Key是否已过期。
- **404错误（模型不可用）：** 可能你用了官方模型名，但千聚暂时没上架。去平台搜索确认模型名称。
- **503错误（服务不可用）：** 检查网络环境是否在国内直连，有时网络波动会导致丢包，重试即可。
