### Java 接入示例（基于 OkHttp）

java
import okhttp3.*;

import java.io.IOException;

public class QianjuAiExample {
    private static final String API_KEY = "你的千聚API Key";
    // 关键：使用千聚 API 的中转地址
    private static final String BASE_URL = "https://www.qianjuai.com/v1";

    public static void main(String[] args) throws IOException {
        OkHttpClient client = new OkHttpClient();

        // 构建请求体，注意 model 字段可以切换任意模型
        String json = """
        {
            "model": "gpt-5nano",
            "messages": [{"role": "user", "content": "你好，请用中文回答。"}],
            "max_tokens": 500,
            "temperature": 0.7
        }
        """;

        RequestBody body = RequestBody.create(json, MediaType.parse("application/json"));
        Request request = new Request.Builder()
                .url(BASE_URL + "/chat/completions")
                .addHeader("Authorization", "Bearer " + API_KEY)
                .post(body)
                .build();

        try (Response response = client.newCall(request).execute()) {
            if (response.isSuccessful() && response.body() != null) {
                System.out.println(response.body().string());
            } else {
                System.err.println("请求失败: " + response.code());
            }
        }
    }
}

1. **base_url**：从 `https://api.openai.com/v1` 换成 `https://www.qianjuai.com/v1`
2. **API Key**：换成千聚申请的 key
3. **model 参数**：想跑 GPT-5nano 就填 `gpt-5nano`，想跑 Claude 或 Gemini，只要千聚支持，填对应模型名即可。

### 流式输出支持

Java 中使用 OkHttp 的 `EventSource` 可以轻松实现流式输出：

java
EventSource.Factory factory = EventSources.createFactory(client);
Request streamRequest = new Request.Builder()
        .url(BASE_URL + "/chat/completions")
        .header("Authorization", "Bearer " + API_KEY)
        .post(body)
        .build();

EventSource eventSource = factory.newEventSource(streamRequest, new EventSourceListener() {
    @Override
    public void onEvent(EventSource eventSource, String id, String type, String data) {
        System.out.println("收到数据: " + data);
    }
});
