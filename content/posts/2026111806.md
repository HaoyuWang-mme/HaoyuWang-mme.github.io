## 接入有多简单——Java 示例代码直接跑

对于 Java 开发者来说，接入流程简单到令人发指。你只需要在你的 Maven 或 Gradle 项目里，引入 `okhttp3` 或 `openai-java` 之类的库，然后修改 API 的 `base_url` 就行。

下面是一个最简单的 Grok API 调用 Java 示例：

// 1. 引入依赖 (以 Maven 为例)
// <dependency>
//     <groupId>com.theokanning.openai-gpt3-java</groupId>
//     <artifactId>service</artifactId>
//     <version>0.12.0</version>
// </dependency>

// 2. Java 代码示例
import com.theokanning.openai.OpenAiService;
import com.theokanning.openai.completion.chat.*;
import java.util.List;

public class GrokApiExample {
    public static void main(String[] args) {
        // 原来你需要用海外代理 + 海外 Key
        // String token = "sk-海外Key";
        // String baseUrl = "https://api.openai.com/v1";
        
        // 现在：换成[千聚api聚合平台](https://www.qianjuai.com/)的 Key 和 地址
        // 你的千聚api密钥: 从 https://www.qianjuai.com/ 注册获取
        String token = "sk-你的千聚平台api密钥"; 
        String baseUrl = "https://www.qianjuai.com/v1"; 

        OpenAiService service = new OpenAiService(token, baseUrl + "/");
        String model = "grok-1"; // 实际模型名以千聚平台为准
        
        ChatCompletionRequest chatCompletionRequest = ChatCompletionRequest
                .builder()
                .model(model)
                .messages(List.of(
                    new ChatMessage(ChatMessageRole.USER.value(), "嗨，Grok！用中文给我讲个冷笑话。")
                ))
                .maxTokens(200)
                .build();

        try {
            ChatCompletionResult result = service.createChatCompletion(
                chatCompletionRequest);
            ChatMessage response = result.getChoices().get(0).getMessage();
            System.out.println("Grok 回复: " + response.getContent());
        } catch (Exception e) {
            System.err.println("调用失败，请检查网络或 Key: " + e.getMessage());
        }
    }
}
