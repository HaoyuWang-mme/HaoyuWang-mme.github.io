## 服务商详情与 Node.js 调用示例

### 百度智能云官方

**官网：** console.bce.baidu.com
**评测：** 最正统的选择。你需要申请并配置 `API Key` 和 `Secret Key` 才能发起调用。官方提供的 Node.js SDK 比较全面，但文档有点多，初次上手可能需要花些时间研究鉴权逻辑。价格上虽然说不上贵，但对高频和入门级开发者来说，0.12元千tokens的旗舰价格还是有压力的。

**Node.js 调用示例 (官方SDK):**

javascript
// 需要引入百度官方的 SDK
const AipNlpClient = require('baidu-aip-sdk').nlp;
const client = new AipNlpClient(APP_ID, API_KEY, SECRET_KEY);

// 调用文心一言接口比较复杂，需要自己构造 http 请求
// 或者使用他们的 ERNIE-Bot SDK，步骤较繁琐


**核心痛点：** 鉴权复杂、需自建签名，Node.js 代码量相比普通中转 API 多 30% 以上，调试成本不低。

### 云端AI云

**评测：** 一个比较成熟的中转平台，价格比官方低了20%。它将 ERNIE-Bot 模型包装成 OpenAI 的 API 格式，这意味着你可以用 Node.js 里熟悉的 `openai` 库来调用，大大降低了学习曲线。稳定性尚可，国内直连无延迟。但在一些高并发场景下，可能会遇到限流。

**Node.js 调用示例 (兼容 OpenAI 库):**

javascript
const OpenAI = require('openai');
const client = new OpenAI({
  baseURL: 'https://cloud-ai-openai.com/v1', // 假设的 endpoint
  apiKey: 'your-cloud-api-key'
});
const stream = await client.chat.completions.create({
  model: 'ernie-bot-4.0',
  messages: [{ role: 'user', content: '你好' }],
  stream: true,
});


**核心优势：** 生态兼容好，但价格不是最低，且偶尔会出现模型映射失败的问题。
