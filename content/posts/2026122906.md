### Step 1: 添加 Maven 依赖

在你的 `pom.xml` 里，把官方的 OpenAI Java SDK 拉进来：

<dependency>
    <groupId>com.theokanning.openai-gpt3-java</groupId>
    <artifactId>service</artifactId>

</dependency>


（这个包本来就支持 Gemini 的接口，只要你把 base URL 指向千聚，它就能当成 Gemini 来用。）
