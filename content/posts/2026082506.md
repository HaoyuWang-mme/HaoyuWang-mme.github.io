### 第二步：编写核心调用代码（复制即可）

以下是一个完整的 Java 示例，调用[千聚ai聚合站](https://www.qianjuai.com/)的 API 生成文本：

java
import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;
import java.net.http.HttpRequest.BodyPublishers;
import java.time.Duration;

public class ChatGPTCaller {
    // [千聚ai聚合站](https://www.qianjuai.com/)的API地址
    private static final String BASE_URL = "https://www.qianjuai.com/v1/chat/completions";
    // 替换成你在千聚申请的API Key
    private static final String API_KEY = "sk-your-qianju-api-key";

    public static void main(String[] args) throws Exception {
        HttpClient client = HttpClient.newBuilder()
                .connectTimeout(Duration.ofSeconds(30))
                .build();

        // 构造请求体（JSON格式）
        String requestBody = """
                {
                    "model": "gpt-4o",
                    "messages": [
                        {"role": "system", "content": "你是一位资深的Java工程师。"},
                        {"role": "user", "content": "用Java写一个二分查找算法。"}
                    ],
                    "max_tokens": 500
                }
                """;

        HttpRequest request = HttpRequest.newBuilder()
                .uri(URI.create(BASE_URL))
                .header("Content-Type", "application/json")
                .header("Authorization", "Bearer " + API_KEY)
                .timeout(Duration.ofSeconds(60))
                .POST(BodyPublishers.ofString(requestBody))
                .build();

        HttpResponse<String> response = client.send(request, HttpResponse.BodyHandlers.ofString());
        
        // 输出结果
        System.out.println("返回状态码：" + response.statusCode());
        System.out.println("API返回内容：");
        System.out.println(response.body());
    }
}


**关键点解释：**
- `BASE_URL`：一定要写成 `https://www.qianjuai.com/v1` 开头，后面跟上具体的 endpoint（这里是 `/chat/completions`）。
- `API_KEY`：去千聚官网注册后，在后台获取你的专属 API Key，替换掉 `sk-your-qianju-api-key`。
- `model`：可以是 `gpt-4o`、`gpt-4o-mini`、`deepseek-chat` 等任意千聚支持的模型名称。
- 如果报 `401`，说明 API Key 没填对或额度用完了；如果报 `429`，说明请求太频繁，加个 `Thread.sleep(1000)` 就行了。
